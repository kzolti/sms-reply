Summary: Android application that handles SMS auto-replies

Description:
An Android application that handles SMS auto-replies with customizable templates.

Features:
* Customizable reply templates
* Automatic replies to incoming calls/messages  
* Multilingual support (English, German, Spanish, French, Hungarian, Polish, Russian, Chinese)
* Starts on boot and runs as a background service


The app allows users to set up automated responses for SMS messages and incoming calls, 
making it easier to manage communication when you're busy or unavailable.
